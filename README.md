# Análise do Impacto da Inteligência Artificial no Mercado de Trabalho
Objetivo do Projeto
Este projeto tem como objetivo analisar o impacto da Inteligência Artificial generativa no mercado de trabalho contemporâneo, utilizando o NotebookLM como ferramenta principal de apoio à pesquisa, organização e síntese das informações.
A proposta é investigar preocupações relacionadas à automação, identificar equívocos comuns e destacar oportunidades emergentes com o avanço tecnológico.

# Ferramenta Utilizada

O projeto foi desenvolvido com o apoio do NotebookLM, utilizado para:
Analisar documentos e artigos sobre IA e mercado de trabalho
Gerar resumos estruturados
Comparar argumentos entre diferentes fontes
Responder perguntas direcionadas sobre riscos e oportunidades da automação

# Fontes Utilizadas

As seguintes fontes foram inseridas no NotebookLM para análise:
Artigos acadêmicos sobre Inteligência Artificial generativa
Relatórios sobre automação e empregabilidade
Textos e publicações sobre transformação digital no trabalho
